# Sports-Company-Data-Medallion-Architecture
End-to-end data engineering project implementing a Medallion Architecture for an acquired sports company's sales data using Databricks, PySpark, Delta Lake, and dimensional modelling.


This project demonstrates the modernisation of data belonging to a newly acquired sports company.

The parent organisation already operates a structured and streamlined data platform. However, the acquired sports company did not have a standardised data model, automated data-processing framework, or clearly defined analytical data layers.

To address this issue, I designed and implemented a scalable data platform using the Medallion Architecture pattern.

The solution processes sports sales data through Bronze, Silver, and Gold layers and produces an analytical dimensional model containing customer, product, pricing, and order information.
